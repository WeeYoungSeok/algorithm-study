## 📝 Commit Convention
| 태그 | 설명 |
|:---:|:---|
| `feat` | 새로운 문제 풀이 추가 (예: feat: BOJ_1000 A+B) |
| `add` | 풀이 외의 파일 추가 (.gitignore, README 등) |
| `fix` | 코드 수정 및 버그 해결 |
| `refactor` | 코드 리팩토링 (효율 개선) |
| `docs` | 문서 수정 (주석, README 등) |

<br>

## 💡 Mindset
- **Keep Going:** 1일 1커밋을 목표로 합니다.
